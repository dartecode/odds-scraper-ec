# Odds Scraper EC

Sistema de scraping de cuotas deportivas para casas de apuestas de Ecuador.

## Objetivo

Este proyecto fue desarrollado para comparar cuotas deportivas entre diferentes casas de apuestas y detectar mejores oportunidades de apuesta.

## Funcionalidades

- Scraping de cuotas deportivas
- Integración con múltiples bookmakers
- Normalización de equipos y mercados
- Inserción automática en PostgreSQL/Supabase
- Logs por ejecución
- Arquitectura escalable por scraper

## Bookmakers integrados

- Altenar
- SportRadar

## Tecnologías

- Python
- Requests
- Playwright
- PostgreSQL
- Supabase
